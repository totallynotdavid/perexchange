# [pkg]: perexchange-cli

Command-line interface for perexchange (local development tool only).

Install from source:

```bash
cd pkg/cli
uv pip install -e .
```

Available commands:

- `perexchange best-buy` - Show best place to buy USD
- `perexchange best-sell` - Show best place to sell USD
- `perexchange top-buy` - Show top 5 places to buy
- `perexchange top-sell` - Show top 5 places to sell
- `perexchange stats` - Show market statistics
- `perexchange fetch` - Fetch and display all rates
